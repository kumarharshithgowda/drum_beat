# 🥁 Drum Beat Web App

An interactive drum kit web application that allows users to play different drum sounds using keyboard keys or mouse clicks.

---

## 🚀 Features

* 🎹 Play drum sounds using keyboard keys
* 🖱️ Click-based interaction
* 🔊 Realistic drum audio
* 💻 Simple and clean UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📁 Project Structure

```
drum-beat/
│
├── images/        # Drum images
├── sounds/        # Audio files
├── index.html     # Main HTML file
├── index.js       # JavaScript logic
└── styles.css     # Styling
```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/kumarharshithgowda/drum_beat.git
```

2. Open the project folder

3. Run:

* Double click `index.html`
  OR
* Use Live Server in VS Code

---

## 🎮 How to Use

* Press keys like **W, A, S, D, J, K, L**
* Or click on drum buttons
* Enjoy the sounds 🎵
